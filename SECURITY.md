# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Adaptix-Docs, please report it
privately. **Do not open a public GitHub issue.**

- Email: security@adaptixcore.com
- Subject line: `SECURITY: Adaptix-Docs`

Please include:

- A clear description of the issue
- Steps to reproduce (if applicable)
- Affected paths, commits, or branches
- Your assessment of impact

You should receive an acknowledgement within 3 business days. We will
coordinate a fix and disclosure timeline with you.

## Supported Versions

Only the `main` branch of this repository is supported. Older tags and
branches do not receive security fixes.

## Scope

This repository contains the public Adaptix-Docs site. Runtime service
vulnerabilities should be reported against the relevant service repository,
not here.
